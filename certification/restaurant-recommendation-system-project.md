## Description

Restaurant Recommendation System is a FastAPI-powered web application that provides personalized restaurant recommendations using a content-based filtering approach.

Users can search for restaurants and refine recommendations using filters such as budget, cuisine preferences, and minimum ratings. The system also includes an analytics dashboard that visualizes restaurant trends and dataset insights.

The project demonstrates the practical application of Machine Learning, recommendation engines, and modern web development technologies in solving real-world decision-making problems.

This project was completed as part of the Applied Data Science Program conducted by SmartBridge and Google for Developers.
