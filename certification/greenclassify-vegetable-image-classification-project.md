## Description

GreenClassify is an end-to-end Deep Learning web application designed to detect, classify, and count vegetables from uploaded images in real time.

The project utilizes a MobileNetV2-based transfer learning model trained on a vegetable image dataset and deployed through a Flask-powered web application. The solution supports multi-vegetable classification, object counting, and rejection of non-vegetable images to improve prediction reliability.

This project was completed as part of the Artificial Intelligence Program conducted by SmartBridge and Google for Developers.
