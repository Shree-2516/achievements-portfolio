## Description

AWS-hosted Virtual Classroom and Learning Platform is a cloud-based educational platform developed using Flask, MySQL, and AWS cloud services.

The application enables students to register, authenticate securely, access learning materials, and download educational resources online. The platform leverages AWS EC2 for application hosting, AWS RDS for database management, and AWS S3 for scalable file storage.

The project demonstrates practical implementation of cloud-native application deployment, cloud storage management, user authentication, and scalable architecture design.

This project was completed as part of the Cloud Practitioner Program conducted by SmartBridge.
